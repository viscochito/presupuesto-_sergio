# Logo de la Empresa

Coloca aquí el archivo del logo de RH Pisos Industriales.

## Instrucciones:

1. **Nombre del archivo:** `logo.png` (debe ser exactamente este nombre)
2. **Ubicación:** Este archivo debe estar en esta carpeta (`public/logo.png`)
3. **Formato:** PNG (recomendado), JPG también funciona
4. **Tamaño recomendado:** 400x400px o mayor para mejor calidad

Una vez que coloques el archivo `logo.png` aquí, aparecerá automáticamente en:
- La previsualización del presupuesto
- El PDF generado
